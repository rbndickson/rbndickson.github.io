A Yelp clone created to practice Rails and TDD.

## Features:

- Authentication
- User profiles
- Business profiles
- Search businesses by name, location or both
- Add businesses
- Business reviews
- Unit and feature tests

## Techn:

Rails, RSpec, Capybara, PostgreSQL, HAML, Bootstrap
